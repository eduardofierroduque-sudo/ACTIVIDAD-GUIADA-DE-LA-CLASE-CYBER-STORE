# Sistema de Inventario - La Tienda Pulenta

¡Bienvenido! Este es un script de Python diseñado para gestionar el stock de una tienda de tecnología de forma eficiente.

## Desafíos Superados
Este proyecto fue una gran escuela de:
- **Depuración:** Aprendí a identificar errores de indentación y sintaxis.
- **Lógica:** Implementación de bucles y condicionales para alertas de stock.
- **Uso de IA:** El código fue refinado usando agentes de VS Code, Gemini y Claude, logrando un código limpio y funcional.
